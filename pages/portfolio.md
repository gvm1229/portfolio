### 프로젝트 개요

React, Next.js, Supabase (PostgreSQL) 를 사용한 풀스택 프로젝트로, 메이플스토리의 각종 장비 세트의 경매장 가격 변동 내역을 그래프로 볼 수 있고, 원하는 목표치를 달성하기 위한 장비의 최적 비용 계산이 가능한 편의성 사이트.
메이플스토리 게임에서 직접 메모해온 경매장 가격 데이터 및 Nexon Open API 를 사용하여 유저의 장비를 불러와 계산에 사용.

-----

### 프로젝트 배경

메이플스토리에서 가장 중요한 요소 중 하나인 아이템 드랍률과 메소 획득량 수치를 제대로 맞추려면 장비를 통한 방법 밖에 없음.
그러나 자신의 장비를 게임 내에서 일일이 확인하고, 수치를 계산하고, 무슨 장비를 얻어야 할지 찾아내고, 경매장 가격까지 찾기엔 너무나도 번거로움.
그러므로 이 과정 중 대부분을 한눈에 볼 수 있게 하거나 자동화하여 불편함을 해소하고자 사이트를 따로 만들고자 함.

### 프로젝트 개요

React, Next.js, MongoDB 를 사용한 풀스택 프로젝트로, 카러플 관련 정보, 팁, 카러플 컨텐츠를 만드는 유튜버들 중 추천 채널과 추천 영상 모음을 전달하는 정보성 사이트.
각 포스트는 Markdown 을 사용한 포스트로, header 를 설정에 맞춰 작성한 후 단일 md 파일을 업로드하면 바로 포스트로서 등록되는 방식.
추천 영상은 직접 선정한 영상들을 Google API 를 통해 받아와 DB 에 등록한 것들.

-----

### 프로젝트 배경

카트라이더: 러쉬플러스 (일명 카러플) 는 중국 서버와 함께 서비스되고 있어 한국 서버와 비교하면 약 10개월 (5시즌) 이 앞서있어 미래시가 존재함.
그럼에도 나무위키 페이지 및 커뮤니티가 빈약하여 향후 시즌에 무엇이 나오는지에 대한 정보는 중국 사이트를 찾아보지 않는 이상 얻기 힘들다는 불편함이 있음.
이 불편함을 해소하고자 공략 사이트를 따로 만들고자 함.
