## 주요 프로젝트
총 2년 반 동안 임했던 프로젝트 중 일부를 추려봤습니다.

프로젝트 리드를 맡으며 **팀원들과의 소통과 프로젝트 진행 주도**, 클라이언트와 의견 조율 등 다양한 경험을 할 수 있었으며,
각 프로젝트마다 상이하지만, 주로 수행한 역할은 대체적으로 **효율적인 계획 수립, 효율적인 로직 구현, Production 환경 배포** 였습니다.

대외비 프로젝트들이기 때문에 코드는 없지만, 링크된 상세 페이지에 각 프로젝트의 배경과 기술적인 면의 작동 방식을 기록해두었습니다.

### 주식회사 글로벌브릿지 (2023.05 \~ 2025.05)

* **인천광역시 ITS 구축: 통행/신호 API 서버 개발 (2024.08 \~ 2025.04)**

>  **프로젝트 개요:**
>  현실 교차로 데이터를 받아 디지털 트윈 시뮬레이션을 구현의 일환으로 시뮬레이션에 들어가는 차량 데이터를 가공하는 API 서버를 만들고자 하는 프로젝트.

>  **핵심 역할:** 프로젝트 리드, 핵심 로직 고안, 효율적인 개발 계획 수립

>  **참가 인원:**
>  * **프로젝트 리드: 본인 1명 (기여도 80%)**
>  * 협업 및 보조: 1명 (20%)

>  **주요 성과:**
>  * FastAPI + Pandas 기반의 대규모 교통 데이터 처리 및 DLR SUMO 기반의 실시간 시뮬레이션 구현
>  * 시간이 많이 소요되는 반복 작업인 데이터 처리를 커스텀 Mapper 를 작성하여 소요 시간 1\~2시간에서 1\~2분으로 단축
>  * 스크립트를 사용한 배포 반자동화 (원클릭 배포)
 
>  **사용 언어 및 기술:** Python · FastAPI · Pandas · DLR SUMO · Tibero DB · PostgreSQL DB · Docker

  [프로젝트 상세](https://github.com/gvm1229/portfolio/blob/master/pages/%EA%B2%BD%EB%A0%A5%EA%B8%B0%EC%88%A0%EC%84%9C.md#%EA%B2%BD%EB%A0%A51-%EC%A3%BC%EC%8B%9D%ED%9A%8C%EC%82%AC-%EA%B8%80%EB%A1%9C%EB%B2%8C%EB%B8%8C%EB%A6%BF%EC%A7%80)
 
* **AI 모델 파일 관리 시스템 (2024.04 \~ 2024.07)**

>  **프로젝트 개요:**
>  기존에 NAS 에 이리저리 흩뿌려져 있던 사내 AI 학습 모델 파일들을 체계적이고 편하게 관리할 수 있도록 하는 파일 관리 시스템을 개발.

>  **핵심 역할:** 프로젝트 리드, 프런트엔드 디자인 및 개발, 각종 공정의 자동화를 통해 효율 향상 도모

>  **참가 인원:**
>  * **프런트엔드 리드 & 백엔드 보조: 본인 1명 (기여도 70%)**
>  * 백엔드 리드 & 프런트엔드 보조: 1명 (30%)

>  **주요 성과:**
>  * Synology NAS API 기반의 파일 업로드, 분류, 중복 차단 웹 시스템 개발
>  * 최소 1개, 최대 5개의 항목을 채워넣는 것만으로 자동으로 적절한 위치에 파일을 저장
>  * Storybook 을 활용하여 웹 시스템 구성 컴포넌트 관리
>  * 스크립트를 사용한 배포 반자동화 (원클릭 배포)
 
>  **사용 언어 및 기술:** TypeScript · React · Next.js · Tailwind CSS · Radix UI · Zustand · NestJS · PostgreSQL DB · Synology NAS API · Docker

  [프로젝트 상세](https://github.com/gvm1229/portfolio/blob/master/pages/%EA%B2%BD%EB%A0%A5%EA%B8%B0%EC%88%A0%EC%84%9C.md#%EC%82%AC%EB%82%B4-%EC%97%85%EB%AC%B4-%ED%9A%A8%EC%9C%A8-%EA%B0%9C%EC%84%A0-%ED%94%84%EB%A1%9C%EC%A0%9D%ED%8A%B8-ai-%EB%AA%A8%EB%8D%B8-%ED%8C%8C%EC%9D%BC-%EA%B4%80%EB%A6%AC-%EC%8B%9C%EC%8A%A4%ED%85%9C-%EA%B0%9C%EB%B0%9C)

### Heyboss Technologies LLC.

* **클라이언트 홈페이지 개발 (2022.10 \~ 2022.12)**

>  **프로젝트 개요:**
>  자영업자들의 홈페이지를 제작, 배포, 유지보수

>  **핵심 역할:** 프로젝트 일원으로서 프런트엔드 개발, 프로토타입 및 완성된 웹사이트 배포

>  **참가 인원:**
>  * 프로젝트 리드 및 사수: 1명 (기여도 60%)
>  * **프로젝트 보조 및 프런트엔드: 본인 1명 (기여도 40%)**

>  **주요 성과:**
>  * 클라이언트 2~4명에 대한 웹사이트들을 동시 진행, 사내 디자인 솔루션으로 커버 가능한 만큼 빠르게 개발
>  * 재직 기간 동안 총 6개 웹사이트 개발에 참여, 모두 성공적으로 런칭 및 긍정적 피드백 받음
>  * 사내 디자인 솔루션에 없는 맞춤형 기능과 디자인 요구사항이 다수 접수, 이에 맞춰 커스텀 컴포넌트 추가 개발 및 Storybook 을 사용한 테스트 및 리뷰 프로세스 신규 수립
>  * 개발된 커스텀 컴포넌트를 모듈화하여 향후 프로젝트를 위한 기반을 다짐
>  * AWS S3 를 사용해 정적 웹사이트 배포, 과정을 스크립트로 자동화함으로써 새로운 변경사항을 업로드 시 빠른 대응 가능하도록 함
 
>  **사용 언어 및 기술:** TypeScript · React · Tailwind CSS · Amazon S3

  [프로젝트 상세](https://github.com/gvm1229/portfolio/blob/master/pages/%EA%B2%BD%EB%A0%A5%EA%B8%B0%EC%88%A0%EC%84%9C.md#%ED%81%B4%EB%9D%BC%EC%9D%B4%EC%96%B8%ED%8A%B8-%ED%99%88%ED%8E%98%EC%9D%B4%EC%A7%80-%EA%B0%9C%EB%B0%9C-%EB%B0%8F-%EB%B0%B0%ED%8F%AC)

>  **프로젝트 개요:**
